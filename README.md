# Glorious Linux Link 🖱️

A lightweight **Python & PyQt6** utility for managing the **Glorious Model D Wireless** on Linux.

### 🌟 Features
* **Real-time Battery Monitoring:** (Via Wireless Dongle).
* **DPI Stage Switching:** Change sensitivity presets.
* **Linux Native:** Built and tested on **Garuda Linux**.

### 🛠️ Setup
1. **Install Dependencies:**
   sudo pacman -S python-pyqt6 python-hidapi
2. **Permissions:**
   Apply the udev rules provided in the repo.
3. **Run:**
   python main.py
